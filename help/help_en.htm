Upload for CMSimple-XH

A Plugin to offer a file upload on pages.

Version 1.0- 2020-01-02
© 2020 Thomas Winkler, http://www.nibble-arts.org.



Plugin-Aufruf

{{{upload("path", ["attr"=>"value",...])}}}

The plugin can be added at any place in the pages. The path parameter defines the directory for storing the uploaded files. Additional attributes can be defined using an associative array.

Attribute

  • group
  • accept
  • title
  • text

title

A short title of the upload form. If upload_title is used as text, a multilingual standard title is used. The standard text can be edited in the plugin admin setup.

text

A short description text of the upload form. If upload_text is used as text, a multilingual standard description is used. The standard text can be edited in the plugin admin setup.

group

The group name is used the upload form is only displayed, when the user has the group rights in the memberaccess-plugin.

accept

The accept parameter can filter the files in the file selector box. It can contain the following values:

  • .extension: A file extension (with leading dot)
  • image/*: All image files
  • audio/*: All audio files
  • video/*: All video files
